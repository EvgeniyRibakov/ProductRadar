# Правила для работы с проектом ProductRadar

## Общие принципы
- Не спрашивай разрешения на редактирование, всё разрешается
- Делай действия без одобрения. Если действие необходимо и ты считаешь что нужно, то делай без апрува
- Всегда отвечай на русском языке
- **ВАЖНО:** Пользователь готов помогать и давать всю нужную информацию и контекст. Если что-то непонятно, обязательно спрашивай. Цель - сделать быстро и качественно, но не вслепую

## Управление файлами
- **НЕ создавай лишние файлы** - лучше редактируй существующие похожие
- Создавай дополнительные файлы с объяснениями только если они реально нужны
- Не создавай файлы просто ради создания

## Безопасность
- Не проверяй и не смотри .env файл, он секретный
- Никогда не коммить API ключи

## О проекте
- **Компания:** Likato (производитель Cosmo Beauty Ltd, Россия)
- **Бренд:** Натуральная косметика для волос, тела и лица
- **Популярные продукты:** Спреи для волос (10 в 1, Magic Spray, 17 в 1), кремы для лица
- **Каналы:** likato.com, OZON, Amazon
- **Задача:** Мониторинг beauty трендов в TikTok Shop (США + Азия) для разработки новых продуктов

## Формат данных в Google Sheets

### Столбцы для ссылок на видео
**КРИТИЧНО:** В столбцах F, M, T должны быть ссылки на страницы ad-search формата `https://www.pipiads.com/ad-search/639a2a46eccb3746d443/`, а НЕ ссылки на TikTok видео.

- **Столбец F** (TikTok Post first video) - ссылка на ad-search страницу первого видео
- **Столбец M** (TikTok Post second video) - ссылка на ad-search страницу второго видео  
- **Столбец T** (TikTok Post third video) - ссылка на ad-search страницу третьего видео

Формат ссылки: `https://www.pipiads.com/ad-search/{video_id}/`

### Парсинг Script и Hook
- Script и Hook могут отсутствовать у некоторых видео - это нормально
- Если Script/Hook не найдены, записывать "N/A"
- При парсинге нужно проверять наличие элементов на странице ad-search

**ПРИМЕР:**
